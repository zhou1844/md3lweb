v1.3.8 最新 2026-05
  • 新增支持 OptiFine 全自动静默安装，无需用户手动操作，安装完成后自动跳转至版本列表
  • 新增侧边栏加入实时日志入口，可在通用设置中开启，支持搜索过滤、仅显示错误、自动滚动与清空
  • 新增更多 UI 自定义选项：侧边栏宽度、全局圆角半径、紧凑模式等
  • 优化重构 ModDetailScreen、VersionDetailScreen 等页面为 MD3 风格，采用 LazyColumn 统一滚动
  • 修复OptiFine 安装完成后进度指示器仍转圈的问题
  • 修复继承父版本（inheritsFrom）的版本启动时 natives 缺失导致 no lwjgl 崩溃的问题
  • 修复日志界面 LazyColumn key 碰撞导致崩溃的问题
  • 修复版本列表右侧"安装"徽章视觉问题，改为简洁箭头图标
v1.3.6 次要版本 2026-05
  • 新增启动界面左侧底部替换为玩家游戏数据仪表盘，展示 Java / 基岩版启动次数、圈占比图、重崩次数、最长单次等统计
  • 新增游戏数据持久化到 data/stats.json，重启后保留历史数据
  • 优化启动界面左右分区采用黄金比例（左 61.8% / 右 38.2%），信息展示更宽敌
  • 优化页面切换动画改为 MD3 Shared Axis Z 风格，进退场采用不同时长曲线更自然流畅
  • 优化下载中心 Java / 基岩版 Tab 图标更改，与全局 MD3 风格一致
v1.3.5 次要版本 2026-05
  • 新增任务栏图标修复，从 ICO 提取 256×256 PNG，main() 入口最早处设置，彻底解决显示 Java 默认图标问题
  • 新增安装器打包脚本 build-installer.bat 自动读取 AutoUpdater.kt 中的版本号，无需手动维护
  • 优化EXE 图标替换为包含 256/128/64/48/32/16 六个尺寸的完整 ICO，文件管理器超大图标视图清晰显示
  • 修复安装器 DisableDirPage=no，重装时强制显示目录选择页，解决无法更改安装路径问题
  • 修复build-installer.bat 改为纯 ASCII 编码，解决中文字符导致 cmd.exe 闪退问题
  • 优化所有数据迁移至启动器同目录 data 文件夹，首次启动自动从旧版 ~/.md3l 迁移
